## Как загрузить на GitHub Pages
1. Создай или открой репозиторий.
2. Загрузи все файлы из этой папки в корень репозитория.
3. Открой `Settings` → `Pages`.
4. Выбери:
   - `Deploy from a branch`
   - branch: `main`
   - folder: `/ (root)`
5. Сохрани и подожди 1–3 минуты.
6. Открой выданную ссылку с телефона.

## Назначение
Этот пакет нужен именно для дневного просмотра с телефона:
- ловить кривые отступы
- смотреть мобильную логику
- находить слабые блоки
- отмечать, какие страницы нужно переделывать

## Важно
Это не production-frontend и не связка с backend.
Это чистый статический visual preview.
